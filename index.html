DevData

A curated listing of quantitative data sources about software developers and the hardware they use: language preferences, editor usage, ecosystem trends, Moore's Law, and more.

Surveys

Stack Overflow Developer Survey

Annual survey of 90,000+ developers covering languages, frameworks, databases, tools, salaries, and work preferences. One of the largest and most widely cited developer surveys. Raw data is downloadable for custom analysis.

JetBrains Developer Ecosystem Survey

Annual survey of 25,000+ developers covering programming languages, tools, technologies, and development practices. Includes detailed breakdowns by language ecosystem.

Annual Languages IDEs Tools

State of JS / State of CSS

Annual surveys tracking the JavaScript and CSS ecosystems. Covers library adoption, satisfaction, awareness, and usage patterns with rich interactive visualizations.

Annual JavaScript CSS Frameworks Interactive

SlashData Developer Nation

Quarterly survey reaching 30,000+ developers worldwide. Covers language adoption, cloud platforms, IoT, machine learning, and developer demographics.

Quarterly Languages Cloud Demographics

HackerRank Developer Skills Report

Annual report on developer hiring trends, in-demand skills, and language preferences based on millions of coding assessments and developer surveys.

Annual Hiring Skills Languages

Hardware Trends: Moore's Law & Processing Power

Karl Rupp's Microprocessor Trend Data

50 years of microprocessor data: transistor counts, clock frequency, single-thread performance (SpecINT), TDP, and number of cores. Raw .dat files with Gnuplot scripts. The most widely cited open dataset for Moore's Law visualizations.

1970–present Downloadable data Transistors Clock speed Cores CC BY 4.0

Our World in Data: Moore's Law

Interactive charts on transistor counts over time, sourced from Rupp & Horowitz. Part of a broader collection covering computing power, memory costs, and storage costs. All charts are downloadable as CSV/PNG.

Interactive Downloadable CSV Transistors CC BY

Wikipedia: Transistor Count

Comprehensive table of transistor counts for CPUs, GPUs, FPGAs, and other chips from the Intel 4004 (1971) to the latest Apple and NVIDIA silicon. Regularly updated by editors.

Continuously updated Transistors CPUs GPUs

TOP500 Performance Development

Tracks FLOPS performance of the world's top supercomputers since 1993. Shows #1, #500, and sum of all systems over time. Biannual list (June and November). The definitive source for peak computing performance trends.

Biannual FLOPS Supercomputers 1993–present

Epoch AI: Notable AI Models

Dataset of compute (FLOPS), training cost, dataset size, and parameters for notable ML models over time. Tracks the exponential growth in AI training compute. Downloadable as CSV.

Continuously updated FLOPS AI/ML compute Downloadable CSV

Epoch AI: ML Hardware Dataset

Detailed specs for 170+ AI accelerators (GPUs, TPUs): FP32/FP16/INT8 FLOP/s, memory bandwidth (GB/s), die size, transistor count, power draw, and price. Downloadable as CSV. CC BY license.

Continuously updated GPU FLOPS Memory bandwidth Downloadable CSV

Wikipedia: FLOPS

Historical tables of CPU and GPU FLOPS performance, including single-precision and double-precision records. Covers consumer hardware, workstations, and GPGPUs from the 1960s to present.

Continuously updated CPU FLOPS GPU FLOPS Historical tables

Hardware Trends: Memory & Storage

John McCallum's Memory Prices

Historical RAM prices per megabyte from 1957 to 2020. One of the most thorough collections of DRAM pricing data, widely cited in academic research. Shows a factor-of-10 price drop roughly every 5 years.

1957–2020 RAM cost per MB Historical

Our World in Data: Memory & Storage Costs

Interactive chart combining historical RAM and storage cost data from McCallum and other sources. Shows cost per megabyte from the 1950s to present for both memory and disk storage.

Interactive Downloadable CSV RAM cost Storage cost

John McCallum's Disk Drive Prices

Historical hard drive and SSD prices per gigabyte from 1956 to present. Companion dataset to the memory prices page, tracking the cost of persistent storage over time.

1956–present HDD cost per GB SSD cost per GB Historical

Wikipedia: Memory Bandwidth

Tables of memory bandwidth by generation (DDR through DDR5, GDDR, HBM). Shows theoretical peak bandwidth for each standard, useful for tracking bandwidth growth over time.

DDR generations RAM bandwidth HBM GDDR

Wikipedia: HDD/SSD Performance History

Historical data on hard drive and solid-state drive transfer rates, including sequential read/write speeds across generations. Covers the transition from HDD to SSD dominance.

Storage bandwidth HDD SSD NVMe

hblok.net: Historical Cost of Memory and Storage

Continues McCallum's work with downloadable CSV files for RAM, magnetic disk, SSD, and flash prices. Includes data points added after McCallum's last update in 2013. CC BY-SA 3.0.

Downloadable CSV RAM cost SSD cost HDD cost

Backblaze: Hard Drive Cost per Gigabyte

HDD pricing trends from a major cloud storage provider running 250,000+ drives. Practical real-world perspective on storage economics, noting where the historical price decline has flattened.

HDD cost per GB Real-world data Analysis

AI Impacts: DRAM Price Trends

Analysis of DRAM price-per-gigabyte trends with data sourced from McCallum and other references. Includes modeling of price decline rates and discussion of methodology.

RAM cost trends Analysis Historical
