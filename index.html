DevData

A curated listing of quantitative data sources about software developers: language preferences, editor usage, ecosystem trends, and more.

Surveys

Stack Overflow Developer Survey

Annual survey of 90,000+ developers covering languages, frameworks, databases, tools, salaries, and work preferences. One of the largest and most widely cited developer surveys. Raw data is downloadable for custom analysis.

Annual Downloadable data Languages Tools Salaries

JetBrains Developer Ecosystem Survey

Annual survey of 25,000+ developers covering programming languages, tools, technologies, and development practices. Includes detailed breakdowns by language ecosystem.

Annual Languages IDEs Tools

State of JS / State of CSS

Annual surveys tracking the JavaScript and CSS ecosystems. Covers library adoption, satisfaction, awareness, and usage patterns with rich interactive visualizations.

Annual JavaScript CSS Frameworks Interactive

SlashData Developer Nation

Quarterly survey reaching 30,000+ developers worldwide. Covers language adoption, cloud platforms, IoT, machine learning, and developer demographics.

Quarterly Languages Cloud Demographics

HackerRank Developer Skills Report

Annual report on developer hiring trends, in-demand skills, and language preferences based on millions of coding assessments and developer surveys.

Annual Hiring Skills Languages

Indices & Rankings

RedMonk Programming Language Rankings

Biannual ranking that combines GitHub activity and Stack Overflow discussion to produce a composite language popularity score. Published since 2010.

Biannual GitHub + Stack Overflow Languages

PYPL PopularitY of Programming Language

Monthly index based on Google Trends data for programming language tutorial searches. Covers languages, IDEs, ORM frameworks, and databases.

Monthly Google Trends Languages IDEs

IEEE Spectrum Top Programming Languages

Annual interactive ranking that combines data from multiple sources including GitHub, Google Search, Stack Overflow, Twitter, and job sites. Customizable weighting.

Annual Multi-source Interactive Languages

Usage & Trend Data

Stack Overflow Trends

Interactive tool showing the percentage of Stack Overflow questions by tag over time. Compare up to 15 tags to see how developer interest in languages, frameworks, and tools has shifted.

Continuously updated Interactive Questions by tag

GitHub Octoverse

GitHub's annual report on the state of open source. Covers top programming languages, contributor growth, repository trends, and global developer activity across 100M+ developers.

Annual Languages Open source Contributors

GitHut 2.0

Quarterly GitHub language statistics derived from pull requests, issues, pushes, and stars. Provides a more granular, continuously updated view of GitHub language usage.

Quarterly GitHub data Languages Interactive

modulecounts.com

Tracks the growth of major package registries over time: npm, PyPI, Maven Central, RubyGems, crates.io, NuGet, Packagist, and more. Simple charts showing total module counts.

Continuously updated Package managers Ecosystem growth

npm trends

Compare npm package download counts over time. Useful for evaluating the relative popularity of competing JavaScript libraries and frameworks.

Continuously updated npm Package downloads Interactive

PyPI Stats

Download statistics for Python packages from the Python Package Index. Provides per-package daily, weekly, and monthly download counts with breakdowns by Python version and system.

Continuously updated Python Package downloads API available

GitHub Innovation Graph

Quarterly data on developer activity, repositories, and languages broken down by country and territory. Downloadable as CSV for research and analysis.

Quarterly By country Downloadable CSV Open data
